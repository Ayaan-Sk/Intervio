# **App Name**: Voice Mockup

## Core Features:

- AI Topic Question Generation: Utilize speech-to-text to capture the user-selected technical topic for the interview. Use this data to make API calls to a Large Language Model, asking for interview questions to be used in the practice interview. This model acts as a tool that generates questions relevant to a specific tech topic.
- Question Presentation: Ask the user questions using text-to-speech.
- Record Answer: Include a microphone to allow users to capture voice and use speech to text models to answer the AI-generated questions.
- AI Answer Analysis: Using Speech-to-text convert verbal responses into written text to assess using an AI Model. This tool enables AI to determine sentiment, rate the quality of responses and extract talking points.
- Feedback System: Give a detailed assessment of performance with the use of a Star Rating System to gauge effectiveness.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke trust and professionalism in the interview context.
- Background color: Light grey (#F0F4F7), near white, to keep the attention on the practice interview content.
- Accent color: Muted teal (#4DB6AC) for interactive elements and feedback highlights, such as buttons, to add a touch of sophistication without being too distracting.
- Body and headline font: 'Inter', a sans-serif font, known for its clean and modern appearance, suitable for on-screen readability.
- Use simple, professional icons for navigation and features, ensuring clarity and ease of use during the interview process.
- Implement subtle animations for feedback and assessment updates to smoothly guide the user through their mock interview progress.
